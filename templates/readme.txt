The file MNI152_T1_0.5_mm_masked.nii is needed for visualisation purposes in the toolbox bot is not required for rsfc calculations.

The original template ("MNI152_T1_0.5mm.nii.gz") can be obtained from the FSL project here: https://git.fmrib.ox.ac.uk/fsl/data_standard/-/blob/master/MNI152_T1_0.5mm.nii.gz
An additional brain mask has been applied to the template used in afxRs (MNI152_T1_0.5mm_masked.nii) for aesthetic reasons.
